## 데일리리포트 : DailyReport
하루를 기록해, 일정을 정리해 보세요. <br>
자주 하는 작업은 규칙을 만들어 클릭 한 번으로 기록하세요. <br>
원하는 기간만큼의 통계를 통해 어디에 많은 시간을 보내는지 체크해 보세요. <br>
**보다 나은 내일에 도움이 될 것입니다.**
